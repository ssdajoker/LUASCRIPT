üéØ What is Gaussian Blobs GSS?

Gaussian Blobs GSS (Gaussian Splatting Syntax) is a domain-specific language for creating smooth, organic shapes using Gaussian functions. It's perfect for prototyping blob-based graphics, soft shadows, and organic UI elements.

üöÄ Quick Start

  1. Write your GSS code in the editor (left panel)
  2. Click the PLAY ‚ñ∂Ô∏è button to execute
  3. Watch your blobs render on the canvas (right panel)
  4. Modify parameters and replay to see changes instantly

üìñ Basic Syntax

Creating a Blob:

gaussian_blob { center = {x, y}, radius = n, smoothness = 0-1, color = {r, g, b} }

Blend Modes:

  • blend_mode "additive" - Colors add together
  • blend_mode "multiply" - Colors multiply
  • blend_mode "screen" - Screen blend effect

üé® Parameters Explained

  • center: {x, y} position on canvas (0-500, 0-400)
  • radius: Size of the blob (10-200 recommended)
  • smoothness: Edge softness (0.0 = hard, 1.0 = very soft)
  • color: RGB values (0-255 each)

üí° Pro Tips

  • Start with 2-3 blobs to understand the basics
  • Use smoothness values between 0.7-0.9 for organic looks
  • Experiment with blend modes for different effects
  • Overlap blobs to create complex shapes
  • Try animating by changing parameters over time
